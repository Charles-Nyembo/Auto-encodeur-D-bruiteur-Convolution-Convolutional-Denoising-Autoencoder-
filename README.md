# Auto-encodeur-D-bruiteur-Convolution-Convolutional-Denoising-Autoencoder-
Ce dépôt présente l’implémentation d’un auto-encodeur débruiteur basé sur des couches convolutionnelles, utilisé pour supprimer le bruit des images. Le modèle apprend de manière non supervisée à reconstruire des images propres à partir d’images bruitées, en capturant des caractéristiques spatiales pertinentes.
